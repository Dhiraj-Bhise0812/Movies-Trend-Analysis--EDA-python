# Netflix Movies Trend Analysis (EDA) 
  

### Project Overview:- 
This project performs an Exploratory Data Analysis (EDA) on a comprehensive movie database to uncover trends in popularity, audience ratings, and genre performance. The goal is to provide data-driven insights into what makes a movie successful in terms of viewer engagement and critical acclaim. 

 📊  Key Questions Explored Using Python, I answered several critical questions: 

•	Popularity Leaders: Which movies achieved the highest and lowest popularity scores?
•	Genre Trends: What are the most common genres, and which ones command the highest average votes? 
•	Release Trends: How has the volume of movie releases evolved over recent years?
•	Correlation Analysis: Is there a relationship between  the popularity of a movie and its average vote count or rating? 

🛠 Tech Stack & Libraries 
•	Language: Python 3.x
•	Data Manipulation: Pandas , NumPy
•	Data Visualization: Matplotlib , Seaborn 
•	Environment: Jupyter Notebook / IPython 

🧩 Technical Workflow 
•	Data Acquisition: 
Importing raw CSV data using pandas.read_csv . 
•	Data Cleaning: * Handling missing values and ensuring data type consistency. Utilizing lineterminator to handle complex string structures in the dataset.
•	Exploratory Analysis: 
Statistical summaries using df.describe() and df.info() . Grouping and aggregating data by Genre and Release Date. 
•	Visual Insights: 
Bar Charts: Displaying top-performing genres by vote average. Scatter Plots: Analyzing the correlation between popularity and vote counts. Distribution Plots: Visualizing the frequency of ratings across the dataset.
 
📈 Featured Insight From the analysis: "Spider-Man: No Way Home" emerged as a leader in both popularity and vote count for the 2021 period, highlighting the dominance of the Action/Science Fiction genres in driving mass audience engagement. 

📂 Project Structure movies_analysis.ipynb : The primary Jupyter Notebook containing the code and visualizations. mymoviedb.csv : The raw dataset used for the analysis.
